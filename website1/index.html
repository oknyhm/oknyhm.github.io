终末之诗 (End Poem) 编辑加合成表 简介 终末之诗（End Poem）会在玩家进入末路之地并杀死末影龙，跳进返回传送门，“通关”“Minecraft”之后伴随着 C418 的“Alpha”配乐在屏幕上出现。其作者是作家 Julian Gough 。 终末之诗以及鸣谢名单于 Java 版 1.0.0 的 Beta 1.9-pre6 版本加入游戏，在 1.20.5 的快照 24w09a 中将泥土背景换成了返回传送门背景。 资源包作者可通过更改 assets/minecraft/texts/end.txt 中的文本来自定义终末之诗的内容。 建议玩家自行在游戏中体验终末之诗的内容（即便英文阅读理解对一些人来说可能有些困难）（正常速度展现完需约 8 分钟）。 游戏中可以使用 Esc 键跳过，每名玩家正常情况下在一个存档仅能阅读一次终末之诗。自 1.17 的 pre4 版本起，可以按住 Ctrl 键加快滚动速度，按住左右两个 Ctrl 键可以使速度达到最快。1.20 的快照 20w17a 版本后，可以用“↑”键向上滚动。 内容 两种颜色区分不同的发言者（注意，观看内容会导致剧透，可能影响游戏趣味，请谨慎观看）（实际也没多少人会认真看）： I see the player you mean. 我看到你所指的那位玩家了。 [Playername]? [玩家名称]? Yes. Take care. It has reached a higher level now. It can read our thoughts. 是的。小心。它已达到了更高的境界。它能够阅读我们的思想。 That doesn't matter. It thinks we are part of the game. 没关系。它认为我们是游戏的一部分。 I like this player. It played well. It did not give up.
